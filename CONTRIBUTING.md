# 设定研究

## 时代背景

游戏设定的现实原型：结合作者居住地和[其相关发言记录](https://discord.com/channels/981511623766331473/981984164347805716/999854144284282880)可推知游戏以**中世纪欧洲**作为世界观原型。

故事发生的具体年代：由游戏剧情中书籍的工业化批量生产可推知游戏年代至少推进到了**15世纪后期**。
